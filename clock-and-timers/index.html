Clock and timers

Some simple examples on how to do a clock or timers.

Updating times.

A few examples on how to present an always-updating clock.

Value Angular expression Description
{[{currentTime}]} {{currentTime}} The value of the $scope variable set to Date.now().
Updates once a second, and includes milliseconds.
{[{currentEpoch}]} {{currentEpoch}} Simply Math.round(parseInt($scope.currentTime) / 1000)
{[{currentTime | date:'medium'}]} {{currentTime | date:'medium'}} Applies a pre-defined date format. More details at the Angular docs.

Countdown timers

Value Angular expression Description
{[{countdownInterval}]} {{countdownInterval}} The countdown is reset to this when it reaches zero.
{[{countdownTicker}]} {{countdownTicker}} Counts the number of seconds to the next reset.
{[{countdownTriggerCount}]} {{countdownTriggerCount}} Number of times the countdown's been triggered.

Relative times

Use the page load timestamp to show how to display relative time. This is the "3 minutes ago" fuzzy time.

Value Angular expression Description
{[{pageLoadDate | date:'medium'}]} {{pageLoadDate | date:'medium'}} The datetime when the page was loaded.
{[{pageLoadedSecondsAgo}]} {{pageLoadedSecondsAgo}} Seconds since when the page loaded.
{[{pageLoadedTimeAgo}]} {{pageLoadedTimeAgo}} Relative time for when the page was loaded. Uses moment.js.
From 0 to 45 seconds, "a few seconds ago"
From 45 to 90 seconds, "a minute ago"
From 90 seconds to 45 minutes, "2 minutes ago... 45 minutes ago"
Other values visit the moment.js docs.
