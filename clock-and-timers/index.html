Clock and timers

Some simple examples on how to do a clock or timers.

Times and Dates

Value Angular expression Description
{[{currentTime}]} {{currentTime}} The value of the $scope variable set to Date.now(). Updates once a second, and includes milliseconds.
{[{currentEpoch}]} {{currentEpoch}} Simply Math.round(parseInt($scope.currentTime) / 1000)
{[{currentTime | date:'medium'}]} {{currentTime | date:'medium'}} Applies a pre-defined date format. More details at the Angular docs.

Countdown timers

Value Angular expression Description
{[{countdownInterval}]} {{countdownInterval}} The countdown is reset to this when it reaches zero.
{[{countdownTicker}]} {{countdownTicker}} Counts the number of seconds to the next reset.
{[{countdownTriggerCount}]} {{countdownTriggerCount}} Number of times the countdown's been triggered.
